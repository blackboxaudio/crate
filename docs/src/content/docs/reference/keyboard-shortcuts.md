Crate supports keyboard shortcuts for common actions. Use these to work faster without reaching for the mouse.

## Global Shortcuts

These shortcuts work from anywhere in the application (when not typing in a text field).

| Action | macOS | Windows/Linux |
|--------|-------|---------------|
| Play/Pause | `Space` | `Space` |
| Focus Search | `Cmd + F` | `Ctrl + F` |
| Clear Selection | `Escape` | `Escape` |
| Select All | `Cmd + A` | `Ctrl + A` |
| Open Settings | `Cmd + ,` | `Ctrl + ,` |
| Toggle Inspector | `Cmd + I` | `Ctrl + I` |
| New Playlist | `Cmd + N` | `Ctrl + N` |
| New Folder | `Cmd + Shift + N` | `Ctrl + Shift + N` |
| Import Files | `Cmd + O` | `Ctrl + O` |

## Tips

### Modifier Keys

| Key | macOS | Windows/Linux |
|-----|-------|---------------|
| Command | `Cmd` | - |
| Control | `Ctrl` | `Ctrl` |
| Option | `Option` | `Alt` |
| Shift | `Shift` | `Shift` |

### Text Input Focus

When your cursor is in a text field (like search):
- `Space` types a space (doesn't trigger play/pause)
- `Cmd+A` / `Ctrl+A` selects all text in the field
- `Escape` clears focus from the field
- Arrow keys navigate within the text (don't control playback)

### Discovering Shortcuts

Many shortcuts are shown in context menus and tooltips throughout the application.
