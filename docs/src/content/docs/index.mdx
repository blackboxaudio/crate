## Why Crate?

<CardGrid stagger>
  <Card title="Speed First" icon="rocket">
    Native performance with Rust backend. No waiting, no lag. Import thousands of tracks and search instantly.
  </Card>
  <Card title="Precision" icon="seti:config">
    Detailed metadata extraction, flexible tagging system, and powerful filtering. Find exactly what you need.
  </Card>
  <Card title="Control" icon="setting">

  </Card>

## Features

<CardGrid>
  <Card title="Library Management" icon="open-book">
    Import audio files with automatic metadata extraction. Handle duplicates intelligently. Track missing files.
  </Card>
  <Card title="Playlists & Folders" icon="list-format">
    Organize your collection with playlists and a hierarchical folder structure. Drag and drop to rearrange.
  </Card>
  <Card title="Flexible Tagging" icon="approve-check-circle">
    Create custom tag categories and tags. Filter with AND/OR logic. Apply tags in bulk.
  </Card>
