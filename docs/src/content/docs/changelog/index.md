## Version 0.1.0

*Initial release*

### Features

- Library management with metadata extraction
- Playlist and folder organization
- Flexible tagging system with categories
- Audio playback with device selection
- Search and filtering
- Dark and light themes
- 10 accent color options
- 5 font choices
- USB device monitoring
- Bulk editing operations
