title: Search & Filtering
description: Find tracks quickly in your library
---

Crate provides several ways to find tracks: quick search, column sorting, and tag filtering.

## Quick Search

The search bar at the top of the track list lets you filter tracks instantly.

### Using Search

1. Click the search bar or press `Cmd+F` / `Ctrl+F`
2. Type your search query
3. Results filter in real-time as you type

### Search Scope

Search matches against:
- Track title
- Artist name
- Album name

All matching tracks appear in the track list. The search applies to the current view (library or playlist).

### Clearing Search

- Press **Escape** to clear the search and show all tracks
- Click the **X** button in the search bar
- Delete all text from the search field

## Sorting

Click column headers to sort the track list.

### Sortable Columns

| Column | Sort by |
|--------|---------|
| Title | Alphabetical by title |
| Artist | Alphabetical by artist |
| Album | Alphabetical by album |
| BPM | Numeric by tempo |
| Key | By musical key |
| Duration | By track length |
| Date Added | By import date |
| Rating | By star rating |
| Color | By assigned color |

### Sort Direction

Click the same column header again to toggle between:
- **Ascending** (A-Z, low-high, oldest first)
- **Descending** (Z-A, high-low, newest first)

An indicator arrow shows the current sort direction.

## Tag Filtering

Use tags to filter your library by attributes. See the [Tagging guide](/crate/user-guide/tagging/) for full details.

### Quick Tag Filter

1. Click a tag in the sidebar
2. The library filters to show only tracks with that tag
3. Click more tags to refine the filter

### Filter Modes

| Mode | Behavior |
|------|----------|
| **OR** | Tracks with ANY selected tag |
| **AND** | Tracks with ALL selected tags |

Toggle modes with the AND/OR button in the tag sidebar.

### Combining with Search

Search and tag filters work together:
1. Select tags to filter by attribute
2. Use search to further refine by text
3. Only tracks matching both appear

## Filtering by View

### Library View

Shows all tracks in your library. Search and tag filters apply to the entire collection.

### Playlist View

Shows only tracks in the selected playlist. Search filters within that playlist.

### Folder View

Shows playlists and subfolders within a folder. Navigate to a playlist to see its tracks.

## Finding Specific Tracks

### By Title or Artist

1. Press `Cmd+F` / `Ctrl+F` to focus search
2. Type part of the title or artist name
3. Results appear immediately

### By BPM Range

1. Click the BPM column to sort
2. Scroll to find tracks in your target range

### By Key

1. Click the Key column to sort
2. Find tracks in compatible keys

### By Attribute

1. Create tags for the attribute (e.g., "High Energy")
2. Tag relevant tracks
3. Filter by that tag

## Tips

### Fast Searching

- Press `Cmd+F` / `Ctrl+F` from anywhere to jump to search
- Press `Escape` to clear search and selection
- Use short, distinctive search terms

### Building Sets

Combine multiple filters:
1. Filter by energy tag (e.g., "Peak")
2. Sort by BPM
3. Search for a specific genre term
4. Result: Peak energy tracks sorted by tempo in your genre

### Finding Related Tracks

1. Note the BPM and key of your current track
2. Sort by BPM or Key
3. Find harmonically compatible tracks nearby
