Playlists let you group tracks together. Folders let you organize playlists into a hierarchy. Together, they provide a flexible system for managing your collection.

## Playlists

A playlist is a named collection of tracks. Unlike tags (which filter your library), playlists contain specific tracks in a specific order.

### Creating a Playlist

There are several ways to create a playlist:

**From the sidebar:**
1. Right-click in the Playlists section
2. Select **New Playlist**
3. Enter a name

**From the toolbar:**
1. Click the menu button in the toolbar
2. Select **New Playlist**
3. Enter a name

### Adding Tracks to a Playlist

**Drag and drop:**
1. Select tracks in the library
2. Drag them onto a playlist in the sidebar
3. Drop to add

**From context menu:**
1. Select tracks
2. Right-click and choose **Add to Playlist**
3. Select the target playlist

### Viewing a Playlist

Click on a playlist in the sidebar to view its tracks. The main area will show only the tracks in that playlist.

### Reordering Tracks

Drag tracks within a playlist to reorder them. The order is saved automatically.

### Removing Tracks from a Playlist

1. View the playlist
2. Select tracks to remove
3. Right-click and choose **Remove from Playlist**

This removes tracks from the playlist only; they remain in your library.

### Renaming a Playlist

1. Right-click the playlist in the sidebar
2. Select **Rename**
3. Enter the new name

### Deleting a Playlist

1. Right-click the playlist
2. Select **Delete**
3. Confirm the action

Deleting a playlist does not delete the tracks; they remain in your library.

## Folders

Folders organize playlists into a hierarchical structure. You can nest folders within folders for deep organization.

### Creating a Folder

**From the sidebar:**
1. Right-click in the Playlists section
2. Select **New Folder**
3. Enter a name

**From the toolbar:**
1. Click the menu button
2. Select **New Folder**
3. Enter a name

### Organizing with Folders

Drag playlists or folders onto a folder to move them inside:

1. Click and hold on a playlist or folder
2. Drag it onto the target folder
3. Release to drop it inside

### Navigating Folders

When you click on a folder, the main area shows its contents as cards:

- **Subfolders** - Shown as folder cards
- **Playlists** - Shown as playlist cards

Click a card to enter that folder or view that playlist.

**Breadcrumb navigation** appears at the top of the folder view, showing your current path. Click any breadcrumb to navigate back.

### Moving Items Between Folders

Drag and drop items to reorganize:

1. Drag a playlist or folder from the sidebar
2. Drop it onto the target folder
3. The item moves into that folder

### Handling Name Conflicts

When moving items, if a name conflict occurs (same name exists in the target folder), Crate offers:

| Option | Effect |
|--------|--------|
| **Overwrite** | Replace the existing item |
| **Merge** | Combine contents (for playlists) |
| **Cancel** | Abort the move |

### Renaming a Folder

1. Right-click the folder
2. Select **Rename**
3. Enter the new name

### Deleting a Folder

1. Right-click the folder
2. Select **Delete**
3. Confirm the action

Deleting a folder also deletes all playlists and subfolders inside it. Tracks remain in your library.

## Example Organization

Here's an example folder structure for a DJ:

```
Playlists/
├── Sets/
│   ├── 2024-01-15 Club Night
│   ├── 2024-02-20 Festival
│   └── 2024-03-10 Private Party
├── Genres/
│   ├── House/
│   │   ├── Deep House
│   │   ├── Tech House
│   │   └── Progressive
│   ├── Techno/
│   │   ├── Peak Time
│   │   └── Melodic
│   └── Drum & Bass
├── Moods/
│   ├── Warm Up
│   ├── Peak Time
│   └── Closing
└── Pending/
    ├── To Sort
    └── New Arrivals
```

## Tips

- Use folders for broad categories (genres, moods, events)
- Use playlists for specific sets or curated selections
- Combine with [tags](/crate/user-guide/tagging/) for flexible filtering across playlists
