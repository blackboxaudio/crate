## Core Features

Import tracks, extract metadata, handle duplicates, and manage missing files. Learn how Crate organizes your collection.

### [Tagging](/crate/user-guide/tagging/)
Build a flexible tagging system with categories and tags. Filter your library with AND/OR logic.

### [Playback](/crate/user-guide/playback/)
Preview tracks with the built-in audio player. Configure output devices.

### [Search & Filtering](/crate/user-guide/search-and-filtering/)
Find tracks quickly with search. Sort and filter your library.

### [Settings](/crate/user-guide/settings/)
Customize themes, fonts, colors, and audio settings.

Monitor USB drives and external storage devices.

Edit metadata, apply tags, and manage multiple tracks at once.

## Interface Overview

Crate's interface is divided into several areas:

| Area | Location | Purpose |
|------|----------|---------|
| Sidebar | Left | Navigation, playlists, tags |
| Track List | Center | Browse and select tracks |
| Inspector | Right (toggleable) | View and edit track details |
| Player | Bottom | Playback controls |
| Toolbar | Top | Common actions |
