title: Tagging
description: Create a flexible categorization system with tags

Tags provide a flexible way to categorize tracks beyond traditional metadata. Unlike playlists, tags filter your library dynamically.

## Understanding Tags

Tags work differently from playlists:

| Feature | Tags | Playlists |
|---------|------|-----------|
| Purpose | Categorize by attribute | Curate specific tracks |
| Selection | Filter library dynamically | Fixed track list |
| Overlap | Track can have many tags | Track can be in many playlists |
| Order | No inherent order | Ordered list |

Use tags when you want to describe attributes (energy level, mood, venue type) that can combine in different ways.

## Tag Categories

Tags are organized into categories. Each category groups related tags together.

### Creating a Category

1. Right-click in the Tags sidebar section
2. Select **New Category**
3. Enter a name
4. Choose a color

Categories have colors for visual identification. The color appears on all tags in that category.

### Example Categories

| Category | Tags |
|----------|------|
| Energy | Low, Medium, High, Peak |
| Mood | Dark, Uplifting, Groovy, Emotional |
| Venue | Club, Festival, Lounge, After-hours |
| Time of Set | Opener, Build, Peak, Closer |
| Status | Favorite, To Review, Overplayed |

### Editing a Category

1. Right-click the category
2. Select **Edit**
3. Change the name or color
4. Save changes

### Deleting a Category

1. Right-click the category
2. Select **Delete**
3. Confirm the action

Deleting a category removes all its tags and removes those tags from all tracks.

## Tags

Tags are individual labels within a category.

### Creating a Tag

1. Right-click on a category
2. Select **New Tag**
3. Enter a name

Or:
1. Right-click in the Tags section
2. Select **New Tag**
3. Choose the parent category
4. Enter a name

### Editing a Tag

1. Right-click the tag
2. Select **Edit**
3. Change the name
4. Save changes

### Deleting a Tag

1. Right-click the tag
2. Select **Delete**
3. Confirm the action

The tag is removed from all tracks that had it.

## Assigning Tags to Tracks

### From Context Menu

1. Select one or more tracks
2. Right-click and choose **Tags**
3. A submenu shows all categories and tags
4. Click a tag to toggle it on/off

### From the Inspector

1. Select a track
2. Open the Inspector (if not visible, press `Cmd+I` / `Ctrl+I`)
3. Find the Tags section
4. Click tags to toggle them

### Tag Toggle Mode

When tracks are selected, the tag sidebar switches to toggle mode:

- Click any tag to apply/remove it from selected tracks
- Tags show their current state:
  - **Filled** - All selected tracks have this tag
  - **Partial** - Some selected tracks have this tag
  - **Empty** - No selected tracks have this tag

### Bulk Tagging

To tag many tracks at once:

1. Select multiple tracks (Shift+Click for range, Cmd/Ctrl+Click for individual)
2. Right-click and choose **Tags**
3. Select tags to apply
4. Tags are applied to all selected tracks

## Filtering by Tags

The power of tags is in filtering. Click tags in the sidebar to filter your library.

### Single Tag Filter

Click a tag in the sidebar to show only tracks with that tag. The library filters immediately.

### Multiple Tag Filters

Click additional tags to add them to the filter. How they combine depends on the filter mode.

### Filter Modes

Toggle between modes using the **AND/OR** button in the tag sidebar header.

| Mode | Behavior |
|------|----------|
| **OR** | Show tracks with ANY selected tag |
| **AND** | Show tracks with ALL selected tags |

**Example with tags:** "High Energy", "Club"

| Mode | Result |
|------|--------|
| OR | Tracks that are High Energy OR suitable for Club |
| AND | Tracks that are BOTH High Energy AND suitable for Club |

### Clearing Filters

Click the **Clear** button in the tag sidebar header to remove all tag filters and show all tracks.

## Tag Display

Tags appear in several places:

### In Track List

Each track row shows its tags as colored chips. This gives you an at-a-glance view of categorization.

### In Sidebar

The tag sidebar shows:
- All categories (expandable)
- All tags within each category
- Tag counts (how many tracks have each tag)
- Current filter state (which tags are active)

### In Inspector

The Inspector shows a track's tags with the ability to add/remove.

## Tips for Effective Tagging

### Start Simple

Begin with a few essential categories:
- Energy (Low, Medium, High)
- Mood (Dark, Light)
- Status (Favorite, To Review)

Add more as your needs evolve.

### Be Consistent

Decide on tag definitions and stick to them. "High Energy" should mean the same thing across your library.

### Use Tags and Playlists Together

- Tags for attributes that help you search
- Playlists for curated selections and set preparation

### Review Regularly

Periodically review tracks tagged "To Review" and update their tags as you learn the music better.
